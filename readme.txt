Automatically inserts excerpts on selected parts of your site and allows you to change things like excerpt length and read more link.

== Description ==
This light-weight plugin gives you more control over your excerpts. Here are its main features:

1. If your theme shows the whole post content on archive pages, the search result page and the blog page then this plugin will help you place excerpts instead of complete posts.
2. It gives you the ability to control the maximum length of excerpts.
3. It also gives you the ability to change the "read more" text.

All these features can be accessed from the control panel. If you like this plugin please give it a good rating.

== Installation ==
1. Setup the plugin with the help of Wordpress' built-in installer.
2. Go to the control panel listed under Plugins and make changes according to your requirements.
3. That's it! Enjoy!
